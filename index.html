Theme Switcher

This is a ThemeSwitcher component implemented with HTML radio group
User choice is remembered in localStorage and used on the next page visit

This is a Tooltip component, hover the items below to see Tooltip in action!

Dropdown

This is a Dropdown select component, click on buttons below to select values from dropdown
Current implementation of Dropdown doesn't support options navigation by Up & Down keys

Autocomplete

This is a Autocomplete component. Start typing in input to see suggestions
You can click on suggestion and its value will be Autocomplete input value
Current implementation of Autocomplete doesn't support options navigation by Up & Down keys

Progress Bar

This is a ProgressBar component, which indicates progress of some operation
At this time, it just updates itself by setTimeout
ProgressBar has 3 states: Active, Stopped and Completed

Modal

This is Modal window component. Click on the buttons below to open modals
Modal has maximum height, so if modal content is too long it becomes scrollable
Modal supports data attribute data-close-on-click-outside, which is by default true
